# nmans

Tool to automate naming of discoveries in _No Man's Sky_ using [Odin's convention][0].

[0]: <https://www.odingaming.com/2018/01/15/no-mans-sky-naming-convention/> "No Man’s Sky Naming Convention – Odin’s Convention"
